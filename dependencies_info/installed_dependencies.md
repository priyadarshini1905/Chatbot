# Project Dependencies

## Installed Dependencies
The following dependencies are used and installed in this project:

- `cors` (^2.8.5)
- `dotenv` (^16.4.7)
- `express` (^4.21.2)
- `groq-sdk` (^0.12.0)
- `multer` (^2.0.2)

## Installation Commands

To install all the dependencies at once (if you already have the `package.json` file), simply run:
```bash
npm install
```

If you are setting up the project from scratch and need to install these exact packages, run the following command:
```bash
npm install cors dotenv express groq-sdk multer
```
